# MVP 使用说明

## 启动

```
1. 打开 Cocos Creator 3.8 编辑器
2. 打开本项目
3. 打开任意场景（或用默认空场景）
4. 按播放 ▶ — GameBootstrap 会自动挂载
```

## 你会看到

- 深绿色背景
- 金色方块 = 左塔（屏幕底部偏左）
- 绿色方块 = 右塔（屏幕底部偏右，需升级安装后出现）
- 彩色方块 = 敌人（从上方生成，向下移动）
- 绿色小方块 = 子弹（炮塔自动发射）
- 金色/绿色半透明矩形 = 攻击范围指示
- 左上角 UI：波次 / 金币 / 经验 / 等级 / 生命（爱心）
- 敌人头顶绿色血条

## 玩法

### 基础循环
- 敌人从上方生成，沿直线向下移动
- 两座炮塔自动找最近的敌人，发射子弹
- 击杀敌人获得 **10 金币 + 15 经验**（不同类型略有差异）
- 敌人到达底部 = 漏怪，扣 **1 点生命**
- 5 条命用完 = 游戏结束

### 双塔系统

| | 左塔（金色） | 右塔（绿色） |
|--|-------------|-------------|
| 伤害 | 50 | 15 |
| 攻速间隔 | 0.6s | 0.25s |
| 初始状态 | 开局可用 | 隐藏，需在升级面板中安装 |

- 左塔开局即可使用
- 右塔需要在升级面板中选择"安装右塔"（花费 25 金币）才会激活
- 安装前右塔节点不渲染、不攻击

### 敌人类型

| 类型 | 解锁波次 | 特征 | 颜色 |
|------|---------|------|------|
| 普通 | 第1波 | 基础属性 | 红色 24×24 |
| 快速 | 第2波 | 血量×0.5，速度×1.8 | 橙红 18×18 |
| 坦克 | 第4波 | 血量×2.5，速度×0.5 | 深红 32×32 |
| 分裂 | 第6波 | 死亡分裂为2个小怪 | 棕色 28×28 |

- 敌人血量 = 80 + 波次×30，速度 = 60 + 波次×8
- 出怪间隔随波次递减，最低 0.4s

### 升级系统
- 经验满后自动升级，弹出 **三选一强化面板**（游戏暂停）
- 升级选项池（Fisher-Yates 洗牌取3个）：
  - 左塔伤害+10、左塔攻速+0.1、左塔伤害+8
  - 右塔未安装时：出现"安装右塔"选项
  - 右塔已安装后：出现右塔伤害+5、右塔攻速+0.15、右塔攻速+0.08
- 每次选择花费 **25 金币**
- 金币不足时按钮变灰，点击任意位置跳过

### 波次递进
- 第一波 10 个敌人，之后每波 +2
- 敌人越来越强（血量 +30/波，速度 +8/波）
- 波次间等待 3 秒
- 第2波起解锁快速敌人，第4波解锁坦克，第6波解锁分裂

### 游戏结束
- 显示结算面板：坚持波次 / 等级 / 金币
- 点击"重新开始"重载场景

## 文件结构

```
assets/scripts/
├── GameBootstrap.ts    ← 唯一入口，自动创建一切 + 游戏逻辑
├── Enemy.ts            ← 敌人：移动 + 减速 + 受伤 + 死亡/逃脱/分裂回调
├── Tower.ts            ← 炮塔：自动索敌 + 射击 + 伤害/攻速加成
├── Bullet.ts           ← 子弹：追踪 + 命中判定
└── ScreenConstants.ts  ← 屏幕常量：720×1280 竖屏设计分辨率
```

## 技术要点

- **零预制体依赖**：所有节点在运行时通过 `box()` + `Graphics` 绘制
- **自动挂载**：场景加载后 GameBootstrap 自动找到 Canvas 并挂载
- **依赖注入**：组件间通过回调函数通信，不依赖场景引用
- **全局暂停**：`GameBootstrap.gamePaused` 控制所有组件的 update
- **无美术资源**：所有视觉元素为彩色矩形，快速验证玩法
