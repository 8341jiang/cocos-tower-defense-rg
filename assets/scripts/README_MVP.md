# MVP 使用说明

## 唯一步骤

```
1. 打开 Cocos Creator 3.8 编辑器
2. 新建场景（或用默认空场景）
3. 创建一个空节点（命名随意）
4. 把 GameBootstrap.ts 挂上去
5. 点击播放 ▶
```

## 你会看到

- 深绿色背景
- 金色方块 = 炮塔（屏幕左侧）
- 红色方块 = 敌人（从右侧不断生成，向左移动）
- 绿色小方块 = 子弹（炮塔自动发射）
- 左上角 UI：波次 / 金币 / 经验 / 等级 / 生命
- 敌人头上绿色血条

## 玩法

- 敌人从右侧生成，沿直线向左移动
- 炮塔自动找最近的敌人，发射子弹
- 击杀敌人获得金币+经验
- 经验够了自动升级，控制台输出升级效果（伤害+5 / 攻速+0.1 / 范围+15）
- 敌人到达左侧 = 漏怪，扣1点生命
- 生命归零 = 游戏结束，控制台输出战绩
- 波次间等待3秒，下一波敌人更强更快更多

## 文件结构

```
assets/scripts/
├── GameBootstrap.ts   ← 唯一入口，自动创建一切
├── Enemy.ts           ← 敌人：移动+血量+死亡
├── Tower.ts           ← 炮塔：自动寻敌+射击
└── Bullet.ts          ← 子弹：追踪+命中扣血
```
